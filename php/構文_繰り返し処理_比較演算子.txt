while構文　繰り返し処理
初期化処理
while(繰り返す条件)　{
 繰り返したい処理
 更新処理
}

$i = 1;
while ($i <= 365) {
  print($i. "\n");
  $i = $i + 1;
}

for構文
for (初期化処理; 繰り返す条件; 更新処理;)　{
  繰り返し処理
}

for ($i=1; $i<=365; $i++) {
  print($i. "\n");
}

■ 比較演算子
A < B　  AがBより小さい
A <= B　 AがB以下
A >= B　 AがB以上
A > B　  AがBより大きい
A === B　AがBと等しい
A !== B　AがBと等しくない
!A       Aではない

$i = $i + 1
$i += 1;
$i++;  //インクリメント(increment=加算)

$i = $i - 1
$i -= 1;
$i--; //デクリメント(decrement=減算)

$i = $i * 5
$i *= 5;

$i = $i / 5
$i /= 5;
